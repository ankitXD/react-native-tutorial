# React Native/Expo Tutorials

Welcome! This folder contains interactive, code-based tutorials teaching React Native and Expo fundamentals.

## 📚 Tutorials Overview

### 1. **Components & Styling** (Beginner)

Learn the basics of building UIs in React Native.

- **Topics**: View, Text, StyleSheet, Flexbox, Pressable
- **What You'll Learn**:
  - Basic React Native components
  - Styling with StyleSheet
  - Creating layouts with Flexbox
  - Making interactive buttons

**File**: `01-components-styling.tsx`

### 2. **State Management** (Beginner)

Master the `useState` hook for managing component data.

- **Topics**: useState, State Updates, Counter, Todo List, Patterns
- **What You'll Learn**:
  - What is state and why it matters
  - Using `useState` hook
  - Managing simple values, objects, arrays
  - Best practices for state management

**File**: `02-state-management.tsx`

### 3. **Hooks & Effects** (Intermediate)

Learn `useEffect` and other hooks for running code at specific times.

- **Topics**: useEffect, useCallback, Dependencies, Cleanup, Patterns
- **What You'll Learn**:
  - Managing side effects with `useEffect`
  - Understanding dependency arrays
  - Cleanup functions for preventing memory leaks
  - Optimizing functions with `useCallback`

**File**: `03-hooks-and-effects.tsx`

### 4. **Lists & Performance** (Intermediate)

Render long lists efficiently with `FlatList`.

- **Topics**: FlatList, Keys, Performance Tips, Patterns
- **What You'll Learn**:
  - Why `FlatList` is better than `map()`
  - Extracting keys properly
  - Performance optimization tips
  - Common list patterns

**File**: `04-lists-and-performance.tsx`

### 5. **Navigation** (Intermediate)

Navigate between screens using Expo Router.

- **Topics**: Expo Router, Routes, Parameters, Deep Linking, Tabs
- **What You'll Learn**:
  - File-based routing with Expo Router
  - Navigating between screens
  - Passing data with route parameters
  - Creating tab and stack navigators

**File**: `05-navigation.tsx`

### 6. **Networking & API Integration** (Intermediate)

Learn to fetch data from APIs and handle network requests.

- **Topics**: fetch(), HTTP Methods, Error Handling, Real APIs, Timeouts
- **What You'll Learn**:
  - Making HTTP requests with fetch()
  - Handling loading and error states
  - Processing JSON responses
  - Best practices for API integration
  - Real-world examples with free APIs

**File**: `06-networking-api-integration.tsx`

## 🎯 How to Use These Tutorials

1. **Start with the Tutorials Index** (`index.tsx`)
   - Shows all available tutorials
   - Click on any tutorial to start learning

2. **Read the Code**
   - Each tutorial has detailed comments explaining concepts
   - Follow along line by line

3. **Interactive Examples**
   - Buttons, inputs, lists you can interact with
   - Try changing values and see what happens

4. **Modify & Experiment**
   - Edit the code and save
   - Hot reload will update your app immediately
   - Break things intentionally to learn!

5. **Take Notes**
   - Write down key concepts
   - Create your own examples
   - Copy useful patterns for your projects

## 💡 Learning Tips

- **Read the comments**: Every important line has comments explaining it
- **Understand before moving on**: Make sure you understand each concept

## 📝 Key Concepts Covered

### React Basics

- Functional components
- JSX syntax
- Component props and children

### React Hooks

- `useState` - manage state
- `useEffect` - run side effects
- `useCallback` - optimize functions
- `useContext` - access global state (covered in State Management tutorial)

### React Native Components

- `View` - container
- `Text` - text display
- `Pressable` - interactive elements
- `ScrollView` - scrollable content
- `FlatList` - efficient lists
- `TextInput` - user input

### Styling

- `StyleSheet.create()` - optimized styles
- Flexbox layout
- Light/Dark themes
- Responsive design

### Navigation

- Expo Router file-based routing
- Stack navigation
- Tab navigation
- Route parameters
- Deep linking

### Performance

- Why `FlatList` over `map()`
- Key extraction
- Memoization
- Memory management
